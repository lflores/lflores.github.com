Based on common tree data "flare", and using D3js library, this example the use of treemap component.

The size of each rectangle is the size of each node, and color is the percentage of total size that cover this node. If you click in a rectangle, you can do a drilldown and dive into data tree

Also, you have a breadcrumb to drill up data
